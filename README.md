## 🚀 HOW TO RUN LOCALLY

### Option 1 — Python (Recommended, no install needed)
Python comes pre-installed on Mac and Linux. Windows users can install from python.org.

```bash
# Navigate to the folder containing these files
cd /path/to/your/folder

# Run the launcher
python3 launch.py
```
The site will open automatically at **http://localhost:8080/onepiece_ultimate.html**

---

### Option 2 — Just Open the HTML File
Simply double-click `onepiece_ultimate.html` in your file browser.

> ⚠️ Note: Some browsers block external images when opening files directly.
> Use the Python server for the best experience with all images loading.

---

### Option 3 — VS Code Live Server
If you have VS Code installed:
1. Install the "Live Server" extension
2. Right-click `onepiece_ultimate.html`
3. Select "Open with Live Server"

---

### Option 4 — Node.js (if you have it)
```bash
npx serve .
# Then open http://localhost:3000/onepiece_ultimate.html
```
